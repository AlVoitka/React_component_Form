Проверка типов с помощью PropTypes

По мере роста вашего приложения вы можете отловить много ошибок с помощью проверки типов. 
Для этого можно использовать расширения JavaScript вроде Flow и TypeScript. Но, даже если вы ими не пользуетесь, 
React предоставляет встроенные возможности для проверки типов. Для запуска этой проверки на пропсах компонента вам 
нужно использовать специальное свойство propTypes:

1.
С версии React 15.5 React.PropTypes были вынесены в отдельный пакет. Так что используйте библиотеку prop-types.

npm i prop-types --save  (save сохранит в список зависимостей)

Этотпакет нужен для отслеживания пропсов которые приходяит в компонент, а точнее теп пропа: строка, число или еще ммного чего

https://ru.reactjs.org/docs/typechecking-with-proptypes.html


2.
import PropTypes from 'prop-types'; ----- импортирую в компонент

class Greeting extends React.Component {
  render() {
    return (
      <h1>Привет, {this.props.name}</h1>
    );
  }
}

Greeting.propTypes = {   ---- к к омпоненту задаю свойство propTypes
  name: PropTypes.string  --- на приходящий пропс вешаем валидацию на строку
};

export default Greeting;

3.
По итогу в консоли будет ошибка если пропс name не будет являтся строкой
