useEffect  - это функция по работе с эффектами, а эффектами называются операции по загрузке данных, использованию 
сторонних модулей, запуск таймаутов и изменение дом-структуры.

хук useEffect объеденил всебе сразу 3 жизненных циклов хука:
- componentDidMount() - вызывается сразу после монтирования (то есть, вставки компонента в DOM)
- componentDidUpdate() - вызывается сразу после обновления. Не вызывается при первом рендере.
- componentWillUnmount() -вызывается непосредственно перед размонтированием и удалением компонента.

1. Импортирую  хук useEffect:

import {Component, useState, useEffect} from 'react';

и совершать действие

    useEffect(() => {
        document.title = `Slide: ${slide}`
    })
